# Project 1

This project is a Streamlit application for analyzing CO2 emissions data.

## Setup

Follow these steps to set up the project environment:

1.  **Create and Activate Conda Environment:**
    If you don't have Conda installed, please follow the instructions [here](https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html).

    ```bash
    conda env create -f environment.yml
    conda activate project1
    ```

2.  **Install Dependencies:**
    The `environment.yml` file handles most dependencies. If any additional packages are needed, they can be installed via pip or conda.

## Running the Application

To start the Streamlit application, ensure your conda environment is activated and run:

```bash
streamlit run app.py
```

The application will open in your web browser.
